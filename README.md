# House Of Viyara26 - Premium Luxury Fashion E-Commerce Website (HTML Version)

## Overview
A modern, elegant e-commerce website built with **HTML5, CSS3, and Vanilla JavaScript** for House Of Viyara26, a premium women's fashion brand.

## ✨ Features
- 🌟 Luxury black, white, gold, and beige color scheme
- 📱 Fully responsive design (mobile, tablet, desktop)
- 💎 Glassmorphism UI effects
- 🎬 Smooth animations and transitions
- 🏠 Homepage with hero slider, categories, featured products, and more
- 🛍️ Shop page with product grid
- 🛒 Shopping cart and checkout pages
- ❤️ Wishlist functionality
- 👤 User account pages
- 📞 Contact and about us pages
- 📧 Newsletter subscription
- 💬 WhatsApp integration
- And much more!

## 🚀 Quick Start Guide

### Option 1: Open directly in browser (simplest!)
1. Download or clone this project
2. Double-click on **index.html** to open it in your browser
3. That's it! Start exploring!

### Option 2: Run a local server (better for development)
If you want to run a local web server for testing, you can use one of these methods:

#### Using Python:
```bash
# Python 3
python -m http.server 8000
```
Then open your browser and go to: `http://localhost:8000`

#### Using VS Code Live Server:
1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html` and select "Open with Live Server"

## 🎨 Customization Tips

### 1. Change Colors
Edit the CSS variables in `assets/css/style.css` (at the top of the file):
```css
:root {
    --primary-black: #0a0a0a;
    --secondary-black: #1a1a1a;
    --white: #ffffff;
    --beige: #f5f1eb;
    --light-beige: #faf7f4;
    --gold: #d4af37;   /* Change this to your brand gold color */
    --light-gold: #e6c547;
    --dark-gold: #b8941f;
    --gray: #666666;
    --light-gray: #e0e0e0;
}
```

### 2. Add Your Logo
Replace `assets/HOV.jpg` with your actual logo file (keep the same filename or update the HTML references).

### 3. Add Product Images
1. Put your product images in the `assets/images/` folder
2. Update the product cards in `index.html` and `pages/categories.html` to use your images

### 4. Update Contact Info
Edit the footer in `index.html` and all page files to update your contact information, WhatsApp number, address, etc.
